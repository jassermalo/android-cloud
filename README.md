# Android Cloud ☁️📱

جهاز أندرويد سحابي يشتغل على GitHub Actions ويمكن الوصول إليه عبر VNC.

## كيفية الاستخدام

### 1. تشغيل الجهاز
- اذهب إلى **Actions** > **Android Cloud**
- اضغط **Run workflow** ← **Run workflow**
- انتظر 5-7 دقائق حتى يبدأ الجهاز

### 2. الاتصال
- في logs الـ workflow ستجد:
  ```
  VNC Address: bore.pub:XXXXX
  ```
- افتح أي برنامج VNC واكتب العنوان

### برامج VNC المجانية
| البرنامج | الرابط |
|----------|--------|
| RealVNC Viewer | https://www.realvnc.com/en/connect/download/viewer/ |
| TigerVNC | https://tigervnc.org/ |
| bVNC (أندرويد) | Google Play |

## المواصفات
- Android 9 (API 28)
- دقة: 1080x1920
- مدة التشغيل: حتى 6 ساعات
